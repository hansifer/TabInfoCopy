# Context menu

Tab Copy adds an item to the browser's web page context menu that can be used to copy the current tab.

<ThemedImage
  style={{
    width:'272px',
    height:'auto',
    marginBottom:'6px',
    filter:'drop-shadow(2px 4px 10px #0000005e)',
  }}
  sources={{
    light: useBaseUrl('/img/context-menu-copy-light.png'),
    dark: useBaseUrl('/img/context-menu-copy-dark.png'),
  }}
  alt="Context menu"
/>

This menu item can be enabled or disabled in [Tab Copy options](./options#show-context-menu).

When this option is enabled, you can also choose whether to show a format selection submenu instead of simply applying the default format.

  style={{

When right-clicking a link or image on a page, the menu presents additional options for copying the link URL or image source.
