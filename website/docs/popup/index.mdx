# Popup

The Tab Copy popup provides [copy buttons](./copy-buttons) and a [format selector](./format-selector).

It can be accessed by clicking the Tab Copy icon ![extension icon](/img/logo-outline-black.png#light-mode-only)![extension icon](/img/logo-outline-white.png#dark-mode-only) in the browser toolbar.
